# PromptKit

PromptKit is a mobile-first AI UI builder built with Flutter.

## Features

- Generate UI components from prompts
- Preview generated HTML/CSS
- Copy, save, and export generated code
- Mobile-first workflow for creators and developers

## Tech Stack

- Flutter
- Dart
- Riverpod
- GoRouter
- WebView Flutter

## Run

flutter pub get
flutter run

## Status

Early development prototype.
